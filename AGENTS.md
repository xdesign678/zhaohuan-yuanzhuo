## 项目
仿《吸血鬼幸存者》的"召唤师"题材 H5 手机网页割草游戏。玩家只走位，召唤 5 种宠物自动作战；元素反应 + 薄猎印连携；宠物 Lv1-5 + 升级三选一 + 轻度局外成长。目标：4 周内手机浏览器可玩、可外发 Demo。

## 最高纪律（单 agent 长跑）
1. 按 **M0→M5 里程碑顺序**推进；每个里程碑结束**必须 `npm run dev` 能跑 + 过验收命令**，才进下一个。
2. **先小步可运行，再加内容**；不要一次铺大功能导致整盘跑不起来。
3. 每次开工先读本文件；改动后保持本文件与现状一致。
4. 每个里程碑完成后 **git 提交（中文 commit）**。

## 技术栈（固定，不要替换）
- Phaser 3 (3.80+) + TypeScript (strict) + Vite 5
- 存档：localStorage（无后端、无账号）
- 音效：jsfxr 程序化生成
- 美术：图像模型生成像素素材 + 后处理 + 纹理图集

## 架构约定
- 目录见 plan.md 第 2 节：`scenes/ core/ systems/ data/ entities/ ui/ storage/ utils/` + `assets/` + `tools/art/`。
- **配置驱动**：宠物/敌人/卡/连携/数值全部在 `src/data/*Defs.ts`；新增内容只改配置 + 可选一个行为函数，**不动主循环**。
- **美术与逻辑解耦**：sprite 一律经 `utils/AssetKeys` 用 key 索引；先几何占位跑通，后换图集，零逻辑改动。
- 主循环顺序：输入→刷怪→宠物索敌/攻击→敌人移动→碰撞(SpatialGrid)→伤害结算→元素反应→经验/升级→波次→UI。

## 编码规范
- TS strict，禁随意 `any`（工具函数除外）。
- **别过度抽象**：`Pet` 就是 `Pet`，别搞 `Entity<Component>` 泛型体操，具体类型优先。
- 单一职责：每个 System 只干一件事。
- 标识符英文、命名清晰；面向人的注释用中文且只在必要处（隐藏约束/坑），不写废话注释。
- 不引入冷门库；Phaser 自带能用就不另装。

## 性能红线（违反必掉帧）
- 碰撞统一走自写 **SpatialGrid (cell 96-128px) + 距离平方判定**；**禁用 Arcade 全量碰撞**。
- 敌人/投射物/宝石/粒子/伤害飘字**全部对象池**，热路径零 `new`。
- 伤害结算 **100ms tick**；宠物索敌 **0.2s 节流 + 锁定粘性**（目标存活且在范围内不重选，防横跳）。
- 视野外实体裁剪；同纹理打图集**合批**；粒子全局上限。
- `DeviceDetect` 帧率自适应降级；敌人软上限**维持刷怪**（不要停刷强化存量怪）。
- 目标：中端机 150-200 怪 ≥30fps，低端降级 ≥24fps。

## 连携防雪崩（必须）
- 同一敌人反应 **1.5s 冷却** + 每帧全局触发数封顶 + 被动扩散出的标记**不再二次引爆**。
- 写 `ElementSystem` 前**先写单元测试**（标记过期 / 双标记触发 / 冷却阻挡 / 同帧不重复）。

## 美术规范（摘要，详见 plan.md 第 8 节）
- 暗色哥特复古**像素风**，固定约 32 色调色板，1px 描边，**透明 PNG**。
- 尺寸：宠物/召唤师 48、杂兵 32、精英 64、Boss 96、拾取 16、UI 64。
- 固定 prompt 模板生成 → **强制后处理**：降采样(nearest) + 调色板量化 + 抠半透明边 → 打图集到 `assets/atlas`，**体积 ≤0.8MB**。
- 动画走**程序化 tween**（呼吸/翻向 flipX/位移/闪白 tint/淡出+粒子），**不做逐帧**。

## 移动端
- 竖屏 FIT；默认**全屏任意拖拽**移动（摇杆为设置备选）。
- canvas `touch-action:none` + `preventDefault`；meta `user-scalable=no`；安全区适配；触摸热区 ≥44px。
- iOS：标题页点击解锁 WebAudio。

## 验收命令
- `npx tsc --noEmit`
- `npm run build`（产物 < 1.5MB）
- `npm run test`（至少连携系统单测）

## 里程碑闸门（done-when）
- **M0** 手机能拖动占位方块、无手势冲突
- **M1** 200 怪 ≥30fps（闸门）且 1 宠能打能升
- **M2** 5 宠 + 连携稳定不雪崩(单测过) + 构筑成型
- **M3** 像素美术风格统一、图集 ≤0.8MB、换图零逻辑改动
- **M4** 局外存档可靠坏档不崩、UI 不崩
- **M5** 中端 150-200 怪 ≥30fps、可外发链接、build < 1.5MB
